CAE 6513 - Analysis of Nonlinear Structures
3
lecture hours
0
lab hours
3
credits
Course Description
This course introduces techniques for the analysis of nonlinear structural systems. Types of nonlinear behavior will be discussed, as well as solution methods. Advanced software will be used to solve nonlinear problems, and students will learn how to validate complicated solutions produced by commercial software.   (prereq: CAE 3511,
CAE 5511
) (quarter system prereq: AE 3211, AE 5210 or CV 5210)
Course Learning Outcomes
Upon successful completion of this course, the student will be able to:
Identify situations when advanced analysis techniques are needed
Solve problems with nonlinear material behavior
Solve problems with nonlinear geometric behavior
Evaluate complex structures with advanced structural software and understand results
Prerequisites by Topic
None
Course Topics
Classical analysis techniques
Cable-supported structures
Arches
Grid structures
Material nonlinearity
Geometric nonlinearity
Adjustments to stiffness matrices for nonlinear effects
Numerical solutions to nonlinear structures
Coordinator
Dr. Edward Sippel

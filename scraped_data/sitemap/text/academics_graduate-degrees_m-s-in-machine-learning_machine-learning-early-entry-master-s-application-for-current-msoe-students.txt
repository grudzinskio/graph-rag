Machine Learning Early Entry Master’s Application for Current MSOE Students
Current MSOE students with junior standing wanting to apply for the Machine Learning Early Entry Master’s program should apply here.
View the Application:
Current MSOE students with junior (60-89 credits) standing wanting to apply for the Machine Learning Early Entry Master’s program should apply here. Please note that only current MSOE undergraduate students should apply here.
If you are not a current MSOE student but would like to apply for the Master of Science in Machine Learning program you should complete the application
here.
Please note, if your current MSOE cumulative GPA is below 3.0, you must include a personal statement that includes discussion of how you plan to meet the additional challenges of graduate coursework so that you can earn at least a minimum of a 3.0 GPA each term moving forward.
Students with a MSOE undergraduate cumulative GPA below 2.7 will not be considered at this time.
If you have any questions, please reach out to
Lucia Kohne
, Director of Graduate Admissions.
